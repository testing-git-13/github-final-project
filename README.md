# github-final-project

Simple Interest CalculatorA simple command-line tool or script that calculates the interest earned on a principal amount over a specific period at a given interest rate. This project is part of the IBM Git and GitHub final project.DescriptionSimple interest is a quick and easy method of calculating the interest charge on a loan or the growth of a deposit. It is determined by multiplying the daily interest rate by the principal by the number of days that elapse between payments.FormulaThe calculation is based on the standard mathematical formula:$$A = P(1 + rt)$$Where:$P$ = Principal amount (the initial deposit or loan amount)$r$ = Annual interest rate (in decimal form)$t$ = Time the money is invested or borrowed for (in years)How to UseTo use the calculator, follow these steps:Input the Principal: Enter the total amount of money borrowed or invested.Input the Rate of Interest: Enter the annual percentage (e.g., enter 5 for 5%).Input the Time Period: Enter the number of years.Calculate: The tool will output the total interest earned and the final balance.ExampleIf you invest 1000 MAD at an annual interest rate of 5% for 3 years:Principal ($P$): 1000Rate ($r$): 0.05Time ($t$): 3Interest: $1000 \times 0.05 \times 3 = 150$Total Balance: 1150LicenseThis project is licensed under the Apache License 2.0.
