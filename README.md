# Simple Interest Calculator

A simple command-line tool that calculates the interest earned on a principal amount over a specific period at a given interest rate. This project is part of the **IBM Git and GitHub Final Project**.

---

## Project Description
This repository contains a documentation-based project for a **Simple Interest Calculator**. Simple interest is a quick and easy method of calculating the interest charge on a loan or the growth of a deposit.

## Formula
The calculation is based on the standard mathematical formula:

**A = P(1 + rt)**

Where:
* **P** = Principal amount (the initial deposit or loan amount)
* **r** = Annual interest rate (in decimal form)
* **t** = Time the money is invested or borrowed for (in years)

---

## How to Use
To use the calculator, follow these steps:

1. **Input the Principal:** Enter the total amount of money borrowed or invested.
2. **Input the Rate of Interest:** Enter the annual percentage (e.g., enter 5 for 5%).
3. **Input the Time Period:** Enter the number of years.
4. **Calculate:** The tool will output the total interest earned and the final balance.

## Calculation Example
If you invest **1000 MAD** at an annual interest rate of **5%** for **3 years**:
* **Principal (P):** 1000
* **Rate (r):** 0.05
* **Time (t):** 3
* **Interest Result:** 150
* **Total Balance:** 1150
